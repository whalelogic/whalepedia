# Crystal Numeric Types

Overview of common numeric primitives and operations.

| Type / Category | Notes | Typical Use |
| --- | --- | --- |
| Integer | Whole numbers | counters, indexes |
| Floating point | Decimal values | measurements, percentages |
| Unsigned (if available) | Non-negative integer range | bitmasks, binary protocols |
| Big number / decimal libs | High precision arithmetic | money, scientific values |

| Built-in Numeric Function | Purpose |
| --- | --- |
| abs | Absolute value |
| min / max | Bound comparisons |
| round / floor / ceil | Numeric rounding modes |

## Example

```text
value = 42
ratio = 3.14
```
