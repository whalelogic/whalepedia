# TypeScript Data Structures

Common structures used for everyday programming tasks.

| Structure | Strength | Typical Use |
| --- | --- | --- |
| Array/List | Fast indexed access | ordered collections |
| Map/Dictionary | Key-value lookups | caches, configuration |
| Set | Unique membership | deduplication |
| Queue/Stack | Ordered processing | jobs, parsing, traversal |

## Example

```text
items -> list
lookup -> map
unique -> set
```
